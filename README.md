# Simple Linked List App

# Data structure
 - linked list

# Variables
 - Node, num1, num2, num3

# Menu
 - 1.) Singly Linked List
 - 2.) Doubly Linked List
 - 3.) Circular Linked List

# Singly Linked List
 - Enter num1: 10
 - Enter num2: 20
 - Enter num3: 30
 - Print 10 20 30

# Doubly Linked List
 - Enter num1: 10
 - Enter num2: 20
 - Enter num3: 30
 - Print 10 20 30

# Circular Linked List
 - Enter num1: 10
 - Enter num2: 20
 - Enter num3: 30
 - Print 10 20 30
